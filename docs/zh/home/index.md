# 简介

SCMeTA 是用于进行单细胞代谢数据处理的开源Python代码平台，由清华大学化学系 [Zhang-lab][lab] 小组开发。

数据处理所用的具体算法可在本小组发表的论文中查询。

源代码已上传至 GitHub [scmeta/scmeta][github].

## 安装

在 PyPI 中，利用以下pip进行安装。

```bash
pip install scmeta
```
