# 源代码

如想更改源代码，可克隆软件源并安装。

```bash
git clone https://github.com/SCMeTA/SCMeTA.git

```

之后使用 pip 或 poetry 进行安装。

```bash
cd SCMeTA
pip install .

## 或使用 poetry
poetry install

```

你也可以直接使用源代码。在资源库的根目录下创建 python 或 Jupyter 笔记本，然后导入软件包。

```python
from SCMeTA import Process

```

更多详情请见 [Using SCMeTA](../quick-start)
